# Smart Expense Tracker üí∞

A modern, responsive web application for tracking personal expenses and income. Built with HTML, CSS, and JavaScript, featuring a beautiful UI and local storage persistence.

## Features

### Core Functionality
- **Add Transactions**: Easily add income or expense transactions with descriptions and amounts
- **Delete Transactions**: Remove transactions from history with a single click
- **Real-time Calculations**: Automatic calculation of total balance, income, and expenses
- **Transaction History**: View all transactions with dates in a scrollable list
- **Data Persistence**: All data is saved locally in the browser using localStorage

### User Interface
- **Modern Design**: Gradient backgrounds, smooth animations, and professional styling
- **Responsive Layout**: Works seamlessly on desktop, tablet, and mobile devices
- **Visual Feedback**: Color-coded transactions (green for income, red for expenses)
- **Interactive Elements**: Hover effects, transitions, and intuitive controls
- **Dark Theme**: Elegant dark gradient background with glassmorphism effects

### Technical Features
- **Local Storage**: Data persists between browser sessions
- **Form Validation**: Input validation for transaction entries
- **Dynamic Updates**: Real-time UI updates without page refresh
- **Date Tracking**: Automatic date assignment for all transactions
- **Unique IDs**: Each transaction has a unique identifier for reliable management

## How to Use

1. **Add a Transaction**:
   - Enter a description in the "Text" field
   - Enter the amount (positive for income, negative for expense)
   - Click "Add transaction"

2. **Delete a Transaction**:
   - Click the "x" button next to any transaction in the history

3. **View Summary**:
   - Check your total balance at the top
   - See income and expense totals in the middle section
   - Scroll through transaction history below

## Technologies Used

- **CSS3**: Modern styling with gradients, flexbox, and animations
